How to verify a mobile number in 3 easy steps

1. Make a sendCode request to send a text to a user's phone

Once the user has entered his or her phone number into your application, call this method to send a text to the user's phone that will contain a code they can type in to prove the phone number belongs to them.

http://localhost:3000/sendCode?developerEmail=yourEmail@example.com&userMobileNumber=412-555-1212&message=Your%20app%20confirmation%20code:

2. Make a checkCode request to verify the number

Once the user has entered in the code, send it back to our server for validation using the checkCode method. This will return a JSON object where validCode is either true if the code is valid and the mobile number is verified, or false if the code is incorrect. http://localhost:3000/sendCode?developerEmail=yourEmail@example.com&userMobileNumber=412-555-1212&code=123456
